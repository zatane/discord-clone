# PREMIUM SYSTEM

## Premium Tier Specifications

### Features Available for Premium Users:
- **Unlimited Message History**: Access to all previous messages in any channel.
- **Enhanced Customization**: Premium users can customize their profiles with unique skins and themes.
- **Increased File Upload Limit**: Upload files up to 50MB, compared to 10MB for free users.
- **Priority Customer Support**: Get faster response times from support.
- **Advanced Analytics**: Access to detailed insights about channel engagement.

### Limitations for Free Users:
- **Limited Message History**: Free users can only see messages from the last 30 days.
- **Basic Customization**: Free profiles come with standard themes only.
- **File Upload Limit**: Maximum file size of 10MB for uploads.
- **Standard Support**: Regular response times for customer inquiries.
- **No Analytics Access**: Limited to basic stats without detailed insights.
